Propósito del sitio

Con el desarrollo de este sitio web se busca adquirir conocimientos en la elaboración de páginas web aplicando las temáticas básicas de html, realizando las diferentes prácticas de acuerdo al buen uso de cada etiqueta, adicionalmente indagaremos más a fondo acerca de los conceptos de CSS en su sintaxis y selectores así como también de la interacción humano - ordenador.

Sintaxis y Selectores CSS

Selectores CSS se utilizan para "find" (o seleccionar) elementos HTML en función de su nombre de elemento, de la identificación, clase, atributo, y mucho más.
